Options

Method

Method determines how time series for crates/cards are computed. For example, if method is "Average", the time series for a crate will be the average of all the CMOS rates for each channel in the crate/card. If method is "Maximum", then the time series will show the maximum CMOS rate for all channels in the crate/card.

Note: This option is not used to combine multiple data points within a single time bin. For example, if there were 10 CMOS readings within a single time bin for a given channel, these are always averaged to produce the CMOS rate for that time bin.
Time Series Step

The time step for a single CSS pixel in the time series.

Crate Map Window

The integration length for data displayed in the crate/card map.

Note: For performance reasons, this is not a rolling integration window, and therefore there will be discontinuous jumps in the data when the window switches from one bin to the next.
